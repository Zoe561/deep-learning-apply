個人簡介和申請動機

非本科系畢業，在北科大 Java 人才培訓班進行半年集訓後，在接案公司磨練四年，現在在永豐金證券擔任系統工程師。 5 年網頁應用軟體工程師，工作主要使用 Angular 、 Node.js (轉ts)完成開發，也曾短暫使用 Java 。

我一直在思考怎麼樣才能賺到更多的錢，而且是以適合我的方式，依我目前的技術發展方向，如果不去博弈業的話，薪資成長幅度好像會越來越低，要靠自己的能力買起房子，我還得努力努力。所以想藉由學習新的技能為我的職涯帶來新的成長，我想要在深度學習這塊站穩腳步，也需要一個有系統化的學習流程，幫助我事半功倍，同時需要同儕激勵我，這裡看起來是個合適的位置。

另外在個人專案的部分，我想要嘗試做一個AI陪伴員，提供情緒價值，或是抒發負面的情緒，現有的 NSFW 過濾制度我並不滿意，且在中文化的部分一直很難讓我有投入感，我想自己做釣竿，不想花錢買魚。

在開始當工程師之前是世新觀光系規劃組(該組已解散)畢業，台北大學都市計劃所肄業，跟高中同學開了一間密室逃脫《迷狐》，經營一年後結束營業，最後評估薪資發展、自身興趣和能力，轉職成工程師。
